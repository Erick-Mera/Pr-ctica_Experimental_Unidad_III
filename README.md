# Pr-ctica_Experimental_Unidad_III
Repositorio académico de la Práctica Experimental N.° 3 de Ingeniería de Requisitos. Contiene la documentación en LaTeX del sistema AquaGest, un sistema para la gestión inteligente de una camaronera, incluyendo revisión del SRS, requisitos refinados, modelo de datos, diagramas ER y UML, DFD, modelo de comportamiento y matriz de trazabilidad.
